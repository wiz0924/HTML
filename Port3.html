Portfolio#3 - Number Systems

What are Number Systems?

Number systems are systems in mathematics that are used to express numbers in various forms and are understood by computers. In addition, there are different types of number systems that we use such as the binary number system, the octal number system, the decimal number system, and the hexadecimal number system. (Cuemath, 2020)

Binary

What Is It?

  • Base 2 (Digits 0 and 1)
  • The foundation of every computer language.

Significance

  • Purest form of computer language
  • Digital Data Storage
  • Networking and Data Transmission

Octal

What Is It?

  • Base 8 (Digits 0-7)
  • Each Octal digit is equal to/ represented by 3 bits

Significance

  • Used in computer systems
  • Uses less digits than other number systems making it more human-readable (Lenovo, 2021) and less prone to error. (Byjus, 2019)

Decimal

What Is It?

  • Base 10 (Digits 0-9)
  • The number system that we use on a daily basis (Cuemath, 2021)

Significance

  • Most widely used numeral system for representing numbers in everyday life, mathematics, and science. (Fiveable, 2025)

Hexadecimal

What Is It?

  • Base 16 (Digits 0-9 & Letters A(10) - F(15) )
  • Each Hexadecimal digit is equal to/ represented by 4 bits

Significance

  • Used to simplify the binary numbering system (Uses of Hexadecimal | Hexadecimal & Character Sets | Computer Science, 2018)
  • Used in computer memory addresses, RGB color codes in graphics (#FF8000), and error codes provided by a device (Makwana, 2025)

Analysis/Reaction

The planet is a vast place with many countries, each with diverse cultures and languages. However, there is one concept that remains universal across the world, numbers.


In this portfolio, we talk about number systems and its different types which are: Binary, Octal, Decimal, and Hexadecimal.


For everyday tasks, the go-to number system has always been the Decimal number system which uses digits 0-9. It is what is used in basic arithmetic, finance, and scientific calculations. On the other hand, we have the other number systems which are not commonly used but are applied in specific fields specially in computer systems. First is Binary, it is the most fundamental number system and it is the language that the computer can understand directly, everything that we input goes through the process of turning into a binary for the computer to interpret it. Next is Octal which is basically a simpler way of writing binary numbers as it is a collection of 3 bits per digit. This allows for readability and accuracy. Lastly, we have Hexadecimal which is similar to Octal but of higher value. To compare, Octal holds/represents 3 bits per digit, while Hexadecimal holds/represents 4 bits per digit. This means that Hexadecimal is more efficient than octal because each digit represents four bits, allowing large binary values to be written compactly. However, because it uses letters A–F, beginners may find it slightly confusing at first.


In summary, it is obvious that among the different types of number systems, Decimal is most used. However, this does not entail that the others are irrelevant as they are applied in various fields that are essential to the world today. The average population may not use the different number systems, but to us who are in the field of technology, each one is just as important as the other.
